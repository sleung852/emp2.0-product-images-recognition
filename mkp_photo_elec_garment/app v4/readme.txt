## Notes for IT Installation ##

1. Install Python 3.6 (preferably Python 3.6.6)

2. Add Python to Environmental Variable - PATH (local is fine)

3a. Then go to the app_v3.py directory
3b. open up Command Prompt (or Terminal) in that directory
3c. type the following code:
	python -m pip install -r requirements.txt --user

4. Ready to use. 

Option 1: (Windows Only)
Just double click app_v3.py

Option 2: (Mac and Windows)
- Open Command Prompt/Terminal on the app_v3.py directory
- type in 'python app_v3.py' to start the program
